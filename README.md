🏠 House Price Prediction ProjectAn end-to-end Machine Learning project to predict house prices using the Ames Housing Dataset. This project implements a production-ready modular architecture, featuring automated feature engineering, model optimization, and a web-based inference UI.🚀 Key FeaturesModular Architecture: Separated concerns for features, preprocessing, training, and inference.Advanced Preprocessing: Automated handling of missing values, scaling, and one-hot encoding using Scikit-Learn Pipelines.Model Optimization: Hyperparameter tuning via GridSearchCV to achieve $R^2 \ge 0.89$.Visualization: Automated generation of Actual vs Predicted, Residuals, and Feature Importance plots.Production Ready: Fully containerized using Docker and served via Flask API.📊 Model PerformanceAfter fine-tuning the Random Forest Regressor, we achieved the following metrics on the validation set:MetricValueR² Score0.89+RMSE~25,000 USDBest ModelRandom Forest (Tuned)🖼️ Visualizations[!TIP]Below are the insights generated by the training pipeline.1.
