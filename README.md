🏠 House Price Prediction Project

An end-to-end Machine Learning project to predict house prices using the Ames Housing Dataset. This project implements a production-ready modular architecture, featuring automated feature engineering, model optimization, and a web-based inference UI.

🚀 Key Features

Modular Architecture: Separated concerns for features, preprocessing, training, and inference.

Advanced Preprocessing: Automated handling of missing values, scaling, and one-hot encoding using Scikit-Learn Pipelines.

Model Optimization: Hyperparameter tuning via GridSearchCV to achieve $R^2 \ge 0.89$.

Visualization: Automated generation of Actual vs Predicted, Residuals, and Feature Importance plots.

Production Ready: Fully containerized using Docker and served via Flask API.

📂 Project Structure

🛠️ Installation & Usage

Local Setup

Clone the repo:

git clone [https://github.com/your-username/house-price-prediction.git](https://github.com/your-username/house-price-prediction.git)
cd house-price-prediction


Install requirements:

pip install -r requirements.txt


Train the model:

python -m src.train


Start Web App:

python app/main.py


Running with Docker

docker build -t house-price-app .
docker run -p 5000:5000 house-price-app
